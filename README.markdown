A simple implementation of the Caesar cipher in 22 lines of Ruby.

    puts Ceaser.encrypt(72,"Hello World! This is an encypted message.") => "BYFFI QILFX! NBCM CM UH YHWSJNYX GYMMUAY."
    puts Ceaser.decrypt(72,"BYFFI QILFX! NBCM CM UH YHWSJNYX GYMMUAY.") => "Hello World! This is an encypted message."
